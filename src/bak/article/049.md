### 0. 安装

ubuntu 系统：

```
sudo apt-get install tig
```

mac

```
brew install tig
```


### 1. 查看所有历史

~~~
cd my_project/
tig
~~~

### 2. 查看 diff

敲 `d`

### 3. 查看某个文件历史

~~~
tig file_name
~~~
